 Relatório  Interativo – Segmentos e Indicadores Financeiros

Este projeto foi desenvolvido em Power BI como parte de um desafio da DIO.  
O objetivo é criar um relatório interativo que permita explorar diferentes perspectivas de mercado e indicadores financeiros.

---

 Estrutura do Relatório

 Página 1 – Análise por Categorias
- Alternar entre Segmento, Produto, País, Semestres e Mês usando parâmetros de campo.
- Visualizar o **lucro por categoria** em gráficos dinâmicos.
- Texto explicativo guia o usuário sobre como interagir com os seletores.

Página 2 – Indicadores Financeiros Dinâmicos
- Alternar entre Lucro (Profit), Vendas (Sales) e Unidades Vendidas (Units Sold).
- Gráfico de colunas mostra a evolução mensal e anual do indicador selecionado.
- Slicer em formato de botões para facilitar a interação.
- Caixa de texto explicativa reforça o storytelling.

---

 Recursos Utilizados
- Parâmetros de campo** para alternar categorias e valores.
- Medidas DAX** (`SUM`) para cálculos dinâmicos.
- Storytelling** com caixas de texto explicativas.
- Formatação visual** para tornar o relatório mais intuitivo.



 Conclusão
Com estas duas visões, oferecemos uma análise completa e flexível:  
- Primeiro identificamos o desempenho por categorias de mercado.  
- Depois exploramos os principais indicadores financeiros de forma dinâmica.  

Essa combinação garante clareza e profundidade, apoiando decisões estratégicas e fortalecendo o planejamento da empresa.
